# Quiz Application - Java

A console-based Quiz Application developed using Java. This project allows users to answer multiple-choice questions within a fixed time limit and calculates the final score based on correct answers.

## Features
- Multiple-choice quiz system
- Timer-based questions
- Score tracking
- Exception handling
- Simple console UI
- Java multithreading implementation

## Technologies Used
- Java
- Collections Framework
- ExecutorService
- Multithreading
- Exception Handling

## How to Run
1. Compile the program:
   javac QuizApplication.java

2. Run the program:
   java QuizApplication

## Concepts Covered
- Object-Oriented Programming (OOP)
- Classes and Objects
- Arrays and ArrayList
- Java Concurrency
- User Input Handling
- Exception Handling
